To run:
1. cd to this foler
2. run make
3. run ./server

4. open new terminal
	cd to this folder
	run ./rw {UNIQUE CLIENT ID} R cooltest.txt {SERVER MACHINE IP}

5. open new terminal
	cd to this folder
	run ./rw {UNIQUE CLIENT ID} W cooltest.txt {SERVER MACHINE IP}
	
6. repeat step 4 and 5 to create as many readers/writers as desired
